# Tài liệu Quy hoạch & Phân tích Dự án (Planning)

Thư mục này chứa các tài liệu phân tích chi tiết về kiến trúc, công nghệ và quy ước lập trình của dự án **L903 Truyện Tranh**.

## Danh mục tài liệu

1. [**Tech Stack**](./tech-stack.md)
   - Phân tích các công nghệ cốt lõi, UI framework, state management và các công cụ hỗ trợ phát triển.

2. [**Coding Conventions**](./conventions.md)
   - Các quy định về đặt tên (naming), cấu trúc component, luồng dữ liệu và tiêu chuẩn code.

3. [**Codebase Architecture**](./codebase.md)
   - Giải thích chi tiết cấu trúc thư mục, luồng xử lý dữ liệu và cơ chế authentication của dự án.

---
*Tài liệu này được tạo ra để hỗ trợ các nhà phát triển nắm bắt dự án nhanh chóng và duy trì tính nhất quán của code.*
